    • NOTE: This chapter does NOT apply if you installed prebuilt toolset executables for BINUTILS, GCC, NEWLIB, and GDB. If you installed prebuilt executables for all of those, proceed to Building RTEMS. If you require a GDB with a special configuration to connect to your target board, then proceed to Building the GNU Debugger GDB for some advice.

    This chapter describes the steps required to acquire the source code for a GNU cross compiler toolset, apply any required RTEMS specific patches, compile that toolset and install it.

    It is recommended that when toolset binaries are available for your particular host, that they be used. Prebuilt binaries are much easier to install.
