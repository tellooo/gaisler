2.4.7: pthread_kill - Send a Signal to a Thread

CALLING SEQUENCE:

#include <signal.h>

int pthread_kill(
  pthread_t thread,
  int       sig
);

STATUS CODES:

ESRCH
The thread indicated by the parameter thread is invalid.
EINVAL
Invalid argument passed.

DESCRIPTION:

This functions sends the specified signal sig to thread.

NOTES:

NONE
