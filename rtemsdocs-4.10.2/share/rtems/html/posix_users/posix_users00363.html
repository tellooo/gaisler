17.4.21: pthread_self - Get Thread ID

CALLING SEQUENCE:

#include <pthread.h>

pthread_t pthread_self( void );

STATUS CODES:

The value returned is the ID of the calling thread.

DESCRIPTION:

This routine returns the ID of the calling thread.

NOTES:

NONE
