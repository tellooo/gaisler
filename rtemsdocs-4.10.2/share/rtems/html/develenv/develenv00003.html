2.1: c/ Directory

  • Directory Structure c/src/ Directory

    • The ${RTEMS_ROOT}/c/ directory was formerly the root directory of all RTEMS source code. At this time, it contains the root directory for only those RTEMS components which must be compiled or linked in a way that is specific to a particular CPU model or board. This directory contains the following subdirectories:

    ${RTEMS_ROOT}/c/src/
    This directory is logically the root for the RTEMS components which are CPU model or board dependent. Thus this directory is the root for the BSPs and the Ada Test Suites as well as CPU model and BSP dependent libraries. The contents of this directory are discussed in the Directory Structure c/src/ Directory section.
