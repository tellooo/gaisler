5.4.2: del_mbx - Delete Mailbox

CALLING SEQUENCE:

ER del_mbx(
  ID mbxid
);

STATUS CODES:

E_OK - Normal completion
E_ID - Invalid ID number
E_NOEXS - Object does not exist
E_OACV - Object access violation

DESCRIPTION:

Specified by the ID, cleans up all data structures and control blocks.

NOTES:

NONE
