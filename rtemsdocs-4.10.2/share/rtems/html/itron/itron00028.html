1.4.12: get_tid - Get Task Identifier

CALLING SEQUENCE:

ER get_tid(
  ID *p_tskid
);

STATUS CODES:

E_OK - Normal Completion

DESCRIPTION:

This system call gets the ID of the issuing task.

If this system call is issued from a task-independent portion, tskid will be FALSE=0.
