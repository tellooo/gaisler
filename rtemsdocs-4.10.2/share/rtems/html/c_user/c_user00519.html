28.4.5: Tail

CALLING SEQUENCE:

rtems_chain_node *rtems_chain_tail(
  rtems_chain_control *the_chain
);

RETURNS

Returns the permanent tail node of the chain.

DESCRIPTION:

This function returns a pointer to the last node on the chain.
