16.3.2: Accessing an Device Driver

The I/O manager provides directives which enable the application program to utilize device drivers in a standard manner. There is a direct correlation between the RTEMS I/O manager directives rtems.io_initialize, rtems.io_open, rtems.io_close, rtems.io_read, rtems.io_write, and rtems.io_control and the underlying device driver entry points.
