Stack Bounds Checker Introduction

RTEMS Ada User's Guide

25.1: Introduction

The stack bounds checker is an RTEMS support component that determines if a task has overrun its run-time stack. The routines provided by the stack bounds checker manager are:

  • rtems.stack_checker_is_blown - Has the Current Task Blown its Stack
  • rtems.stack_checker_report_usage - Report Task Stack Usage
