13.3.1: Creating a Partition

The rtems.partition_create directive creates a partition with a user-specified name. The partition's name, starting address, length and buffer size are all specified to the rtems.partition_create directive. RTEMS allocates a Partition Control Block (PTCB) from the PTCB free list. This data structure is used by RTEMS to manage the newly created partition. The number of buffers in the partition is calculated based upon the specified partition length and buffer size. If successful,the unique partition ID is returned to the calling task.
