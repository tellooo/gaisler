22.3.3: Deleting an Extension Set

The rtems.extension_delete directive is used to delete an extension set. The extension set's control block is returned to the ESCB free list when it is deleted. An extension set can be deleted by a task other than the task which created the extension set. Any subsequent references to the extension's name and ID are invalid.
