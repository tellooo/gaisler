16.2.7: Device Driver Initialization

RTEMS automatically initializes all device drivers when multitasking is initiated via the rtems.initialize_executive directive. RTEMS initializes the device drivers by invoking each device driver initialization entry point with the following parameters:

major
the major device number for this device driver.
minor
zero.
argument_block
will point to the Configuration Table.

The returned status will be ignored by RTEMS. If the driver cannot successfully initialize the device, then it should invoke the fatal_error_occurred directive.
