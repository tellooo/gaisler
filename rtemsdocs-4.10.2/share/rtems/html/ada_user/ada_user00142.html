8.3.5: Canceling a Timer

The rtems.timer_cancel directive is used to halt the specified timer. Once canceled, the timer service routine will not fire unless the timer is reinitiated. The timer can be reinitiated using the rtems.timer_reset, rtems.timer_fire_after, and rtems.timer_fire_when directives.
