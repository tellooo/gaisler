15.3.3: Converting an Address

The rtems.port_external_to_internal directive is used to convert an address from external to internal representation for the specified port. The rtems.port_internal_to_external directive is used to convert an address from internal to external representation for the specified port. If an attempt is made to convert an address which lies outside the specified DPMA, then the address to be converted will be returned.
