13.3.3: Acquiring a Buffer

A buffer can be obtained by calling the rtems.partition_get_buffer directive. If a buffer is available, then it is returned immediately with a successful return code. Otherwise, an unsuccessful return code is returned immediately to the caller. Tasks cannot block to wait for a buffer to become available.
