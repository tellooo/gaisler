2.1: Introduction

The monitor task is a simple interactive shell that allows the user to make inquries about he state of various system objects. The routines provided by the monitor task manager are:

  • rtems_monitor_init - Initialize the Monitor Task
  • rtems_monitor_wakeup - Wakeup the Monitor Task
